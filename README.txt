POS Software

Goal:
This is a personal project intended as an learning exercise, for the purpouse of gaining deeper understanging of programming concepts such as OOP, Database design, Logic and Dataflow, among other things.

Project description:
The software is designed as a POS Solution app, capable of handling basic POS workflow, such as processing transactions, adding and managing customers/employees/items, generating receipts and basing reporting functionality. Also there is a capability simulating workload by automatically generating transaction traffic.

Features:
+ Employee management
+ Customer management
+ Item Management
+ Transaction Management
+ Menu system
- User Accounts
- Application logging
- Error handling
- GUI
- Employee Access levels
- Reporting
- Receipt generation
